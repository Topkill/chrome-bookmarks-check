⭐ 书签哨兵 - 设置

基本设置

开启后，访问网页时自动扫描并标记已收藏的链接
选择右键点击单个链接检查收藏状态时的反馈方式。
选择批量检查链接（右键菜单提取URL、主界面提取URL）时的反馈方式。

URL匹配设置

选择在比较URL时忽略哪些差异（规范化处理）

例如：http://example.com 与 https://example.com 视为相同
例如：example.com/path 与 example.com/path/ 视为相同
例如：EXAMPLE.COM 与 example.com 视为相同
例如：www.example.com 与 example.com 视为相同
例如：example.com#section 与 example.com 视为相同

高级设置

设置系统通知的自动关闭时间，0表示永久存在直到手动关闭
设置单链接弹窗的自动关闭时间，0表示永久存在直到手动关闭
设置多链接弹窗的自动关闭时间，0表示永久存在直到手动关闭
