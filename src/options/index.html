帮助说明

功能介绍

书签哨兵是一个智能的书签管理扩展，能够自动检测网页上的链接是否已被收藏。

使用方法

  1. 安装扩展后，会自动建立书签缓存
  2. 浏览网页时，已收藏的链接会自动标记⭐图标
  3. 右键点击链接或选中文本，可以快速检查是否已收藏
  4. 点击扩展图标查看当前页面的统计信息

性能优化

本扩展使用布隆过滤器技术，即使有数万个书签也能保持快速响应。
